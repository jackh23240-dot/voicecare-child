# 家音英语 (HomeVoice English) - 子女端 WebApp

> 让亲情在英语学习中流淌 🎙️

## 📱 项目简介

家音英语是一款专为子女设计的亲情英语学习应用。通过每日场景化单词学习，子女可以录制温暖的学习声音，打包成"声音包裹"分享给父母家人，让英语学习成为连接亲情的纽带。

## ✨ 核心功能

### 1. 每日灵感卡片 📚
- 类似 Instagram 信息流的卡片设计
- 每天精选 3 个场景化单词
- 涵盖旅行、购物、餐饮、健康等生活场景
- 包含 AI 智能脚本提示

### 2. 长按录音 🎙️
- 长按录音按钮开始录制
- 实时声波动画可视化
- Web Audio API 实现专业录音效果
- 录音时长显示

### 3. AI 智能脚本 💡
- 智能提示如何向家人讲解单词
- 示例："妈，今天教你'Passport'，就是护照的意思"
- 让学习分享更自然温暖

### 4. 派单预览 📦
- 录音完成后预览"声音包裹"
- 可回放录制的音频
- 查看学习内容和时长

### 5. 微信分享 💚
- 生成精美分享卡片
- 一键分享到微信
- 支持复制链接分享

## 🛠️ 技术实现

### 前端技术栈
- **HTML5** - 语义化结构
- **CSS3** - 现代样式和动画
- **JavaScript (ES6+)** - 原生实现，无框架依赖

### 核心 API
- **Web Audio API** - 音频处理和波形分析
- **MediaRecorder API** - 音频录制
- **IndexedDB** - 本地音频数据存储
- **localStorage** - 用户数据和设置
- **Canvas API** - 声波可视化和分享卡片生成

### 特性
- ✅ 响应式设计（手机优先）
- ✅ PWA 就绪（可添加到主屏幕）
- ✅ 离线可用
- ✅ 触摸优化交互
- ✅ 无障碍支持

## 🚀 快速开始

### 本地运行
```bash
# 进入项目目录
cd homevoice-parent

# 使用任意 HTTP 服务器启动
# 方式 1: Python
python -m http.server 8080

# 方式 2: Node.js (需要 http-server)
npx http-server -p 8080

# 方式 3: PHP
php -S localhost:8080
```

### 访问应用
打开浏览器访问 `http://localhost:8080`

### 移动端测试
确保设备在同一网络，访问 `http://你的 IP:8080`

## 📊 单词卡片数据结构

```javascript
{
    id: 1,
    scene: '过海关',           // 场景
    sceneIcon: '🛃',           // 场景图标
    word: 'Passport',          // 单词/短语
    translation: '护照',        // 中文翻译
    pronunciation: '/ˈpæspɔːrt/', // 音标
    example: {
        en: 'May I see your passport, please?',
        zh: '请出示您的护照好吗？'
    },
    aiScript: '妈，今天教你 "Passport"...', // AI 脚本
    category: 'travel',        // 类别
    difficulty: 'easy'         // 难度
}
```

## 🎯 使用流程

1. **打开应用** → 查看今日 3 张灵感卡片
2. **点击卡片** → 进入录音页面
3. **长按录音** → 录制讲解声音（带声波动画）
4. **松开结束** → 自动进入预览页
5. **预览确认** → 回放录音，确认无误
6. **分享发送** → 分享到微信或发送包裹
7. **查看记录** → 在历史记录中回顾学习

## 🌟 特色交互

- **长按录音** - 防止误触，类似微信语音
- **声波动画** - 实时可视化录音状态
- **卡片收藏** - 标记重要单词
- **Toast 提示** - 轻量操作反馈
- **平滑导航** - SPA 单页应用体验

## 🛠️ 开发扩展

### 自定义主题色

### 添加新场景类别
在卡片数据中添加新的 `category` 值。
